# Fullstack Assignment

## Setup & Run (quick steps)

### Backend

cd backend
npm install
cp .env.example .env  # edit with your keys if you want ai
npm run dev            # starts server on 4000

Or run both services with Docker Compose (see backend/README.md).

### Frontend

cd frontend
npm install
npm run dev
